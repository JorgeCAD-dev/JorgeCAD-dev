Estrutura de um documento HTML

       Todo documento HTML deve iniciar com uma tag <!DOCTYPE html> que define o tipo de documento a ser interpretado pelo navegador, em seguida abrimos o documento com uma tag <html>

<!DOCTYPE html> <html>     <body>        <h1>Título</h1>        <h2>Subtítulo</h2>        <p>Parágrafo</p>        <!-- Comentário que não aparecerá no navegador -->     </body> </html>
