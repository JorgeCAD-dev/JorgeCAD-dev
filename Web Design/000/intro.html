O HTML

       HTML é a abreviação de HiperText Markup Language que significa Linguagem de Marcação de Hipertexto, e como o próprio nome diz é uma linguagem de marcação assim como XML, utilizada para descrever páginas da web a serem exibidas em navegadores, e atualmente também para descrever interfaces utilizadas em aplicações com runtimes que permitem execução da páginas web fora do navegador.

       Os arquivos HTML possuem a extensão .html e .htm e podem ser escitos em qualquer editor de textos que serão interpretados pelo navegador. É uma linguagem que suporta:

       Mídia: exibição de conteúdos de áudio, video, gráficos 2D e 3D.

       Eventos: ações do usuário como clique e movimentação do mouse.

       Scripts e Estilos: Pode receber código em CSS para definição de estilo e Javascript para manipular lógica.
