Verificação de Arquivos

       A Verificação de Arquivos serve para localizar arquivos do sistema ausentes ou corrompidos, que interferem no funcionamento do computador. No Windows 10, temos duas etapas para efetuar essas correções, a primeira é a restauração da imagem do sistema e a segunda é a verificação dos arquivos.

1. Clique em Iniciar e digite CMD.

2. Clique com o botão direito sobre o Prompt de Comando e selecione executar como administrador.

3. Digite: DISM.exe /On-line /Cleanup-image /Restorehealth

A Ferramenta DISM (Gerenciamento e Manutenção de Imagens de Implantação) utiliza o Windows Update para recuperar on-line os arquivos danificados.

       Assim que terminar, se necessário abra o CMD como administrador novamente e digite: sfc /scannow. A ferramenta SFC.exe substituirá os arquivos corrompidos do sistema por uma cópia em cache armazenada no diretório do Windows.

A verificação de arquivos e restauração da imagem do Windows deve ser feita quando o computador apresentar comportamento anormal como problemas para abrir programas e erros de DLL e de arquivos do sistema faltantes.
