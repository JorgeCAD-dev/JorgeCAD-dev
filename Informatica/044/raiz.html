A Pasta Raiz

       O diretório raiz do sistema operacional é o Disco Local C:, para acessá-lo basta digitar C: na barra de endereços no Explorador de Arquivos.

       Dentro do diretório raiz temos algumas pastas que são importantes para o funcionamento do computador, como a pasta Windows, que contém o sistema operacional, a pasta Usuários, que contém os arquivos pessoais de cada usuário cadastrado no computador, e as pastas Arquivos de Programas e Arquivos de Programas (x86), que contém todos os programas instalados tanto para a versão 64 bits (x64) quanto os programas de 32 bits (x86).

       Vale ressaltar que os programas não utilizam apenas as suas respectivas pastas de arquivos, mas várias outras, por isso não basta excluir a pasta em que ele está instalado para removê-lo do computador, é preciso fazer a desinstalação adequada.

       Vejamos um pouco sobre estas pastas:

A pasta Arquivos de Programas

       Num computador teremos dois tipos de arquivos, os pessoais que são criados ou salvos por nós e os arquivos do sistema e de programas, que são gerados pelo sistema operacional ou pelos programas no momento da instalação, enquanto o primeiro tipo podemos modificar ou excluir manualmente pelo explorador de arquivos, o segundo só deve ser modificado ou removido utilizando as ferramentas adequadas.

       No diretório raiz (C:), o sistema operacional fica armazenado na pasta Windows, esta pasta está em constante modificação, de acordo com as modificações que fazemos em nosso computador, essas modificações resultam ainda em registros que podem ser acessados pelo Editor de Registros, e ainda geram eventos que podem ser verificados pelo Log de Eventos. O mesmo acontece com os arquivos dos programas que instalamos, e que por padrão são salvos em Arquivos de Programas. Em versões de 64 bits do sistema operacional, haverá duas pastas com esse nome, isso porque os programas de 32 bits suportados pelo computador serão salvos em Arquivos de Programas(x86).

A Pasta do Usuário

       Já os arquivos de usuário, que são salvos por quem está utilizando o computador, ficam localizados na pasta Usuários, que contém os perfis dos usuários cadastrados para utilizar o computador. Você pode cadastrar contas de usuário em Iniciar → Configurações → Contas → Email e Contas.

       Cada perfil da pasta usuário possui uma pasta específica onde ficam armazenadas as pastas Documentos, Imagens, Músicas, Vídeos, Downloads, entre outras. Quando utilizamos o Explorador de Arquivos, na barra da lateral esquerda, o que temos são atalhos para essas pastas. Pela pasta Usuários, é possível acessar as pastas de outros usuários que utilizam o computador, no entanto, caso a conta a qual o perfil pertence possua uma senha, ela será solicitada. O mesmo acontece ao acessarmos uma pasta de um usuário em outro computador da rede cuja conta possua uma senha.
