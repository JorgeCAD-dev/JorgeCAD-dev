Microsoft Edge

       A partir do Windows 10 o navegador padrão da Microsoft é o Edge e nas versões mais recentes o Edge baseado em Chromium, em versões anteriores do Windows o navegador padrão foi o Internet Explorer. Algumas opções de terceiros para navegar na internet são o Google Chrome, o Mozilla Firefox e o Opera. O navegador utilizado nesta apostila é o Edge baseado em chromium que é semelhante ao Google Chrome e já vem instalado junto com o sistema operacional.

       O Microsoft Edge possui uma gama de recursos úteis para quem navega com frequência na internet, como salvar senhas e preenchimentos de formulários, sincronização dos dados com uma conta da Microsoft (a mesma usada no login do computador), sincronização entre outros dispositivos, tradutor embutido para páginas da internet e trechos de texto, coleções de páginas, favoritos, leitor de tela em voz alta, entre outros recursos.
