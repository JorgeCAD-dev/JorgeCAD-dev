Introdução

       Esta página é uma compilação de títulos retirados em ordem do primeiro módulo Curso Completo de Desenvolvimento Web. Este primeiro módulo cobre os principais conceitos de Web Design que são: HTML, CSS e Javascript básico e orientado ao DOM HTML.

       O Web Design é o setor voltado para a criação de páginas Web Estáticas destinadas a exibir conteúdo, como sites, blogs entre outras, é um amplo setor e muito requisitado no mercado. No entanto em módulos avançados deste curso aprenderemos também o desenvolvimento de aplicações web que gerenciam lógica e bancos de dados e APIs da web capazes de receber solicitações do usuário.

       Este material pode ser adquirido impresso na internet ou por meio de seu instrutor, também pode ser adquirido em formato digital (e-book). Este é o material oficial do Curso Completo de Desenvolvimento Web. A venda desta apostila por qualquer outro meio que não seja pelo site citado, por um instrutor autorizado ou diretamente pelo autor do curso, está desautorizada.

       Todas as marcas citadas neste livro estão protegidas por seus próprios direitos autorais. A utilização das mesmas serve unicamente para fim didático. O Autor, não possui qualquer ligação com nenhuma marca citada e não promove o marketing das mesmas.

Mas desenvolver uma metodologia de ensino é bem diferente, a gente só aprende isso com experiência em sala de aula. Isso você não encontra no Youtube. Esse é o diferencial dos cursos disponibilizados por mim, uma metodologia de ensino que mescla experiência de sala de aula, experiência em escrita de livros (pois o autor é também escritor), e experiência no cuidado com os alunos. Tudo para garantir que ao término do curso não tenha apenas ocorrido um processo mecânico, mas para que o aluno possa começar do zero e sair dominando o assunto para se inserir no mercado de trabalho.
