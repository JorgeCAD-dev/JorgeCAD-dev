Um pouco de história

O Microsoft Windows é uma família de sistemas operacionais das quais pertencem o Windows Mobile (utilizado em smartphones e tablets como a linha Surface), o Windows Server e o Windows 11. O Windows foi lançado em 1985 em modo Shell (operado por comandos de teclado). Na época o sistema operacional se chamava apenas MS-DOS.

A primeira versão do Windows com ícones e operada pelo mouse foi o Windows 1.0. É curioso ressaltar que o mouse foi criado pela Xerox, mas os executivos na época riram da invenção e disseram que nunca algo chamado “rato” venderia no futuro. Mas a Apple e a Microsoft investiram na ideia, a Apple lançou o primeiro computador com um sistema operacional baseado em interface gráfica, o Lisa, e a Microsoft lançou o Windows 1.0, que já contava com o Writer, o editor de texto, o Paint, um calendário, uma calculadora e um gerenciador de arquivos. Foi ainda nesse sistema operacional que o conceito de janela foi criado, pois já era possível manter abertos mais de um aplicativo ao mesmo tempo e acessá-los por meio de ícones no rodapé da tela, algo que fez muito sucesso.

Hoje o Windows é o sistema operacional para PCs mais utilizado no mundo, e no segmento de sistemas operacionais só perde para o sistema Android dos smartphones.
