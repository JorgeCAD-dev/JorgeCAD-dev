Sistemas de Cores

Como o mundo do Design Gráfico é um mundo digital, é importante conhecermos os sistemas de cores existentes, o primeiro, e com o qual a maioria das pessoas já está acostumada é o sistema de cores CMYK, pois é o sistema das tintas usadas nos cartuchos de impressão e segue um padrão baseado nas cores primárias. O sistema CMYK é composto pelas cores Cyan (C), Magenta (M), Yellow (Y) e preto (K) que é a cor chave (key).

O sistema CMYK possui variações que vão de 0 a 100, sendo assim, 0,0,0,0 produziria a cor branca, enquanto 100,100,100,100 produziria o preto total.

No sistema CMYK, a combinação 0,0,0,100 produziria apenas um cinza escuro próximo do preto, para produzir o ciano por exemplo, utilizaríamos 100,0,0,0, e o amarelo 0,0,100,0 e assim por diante. Neste sistema temos uma disponibilidade de cores de 100x100x100x100 ou seja 100.000.000 (cem milhões) de cores possíveis.

As escalas de cinza em CMYK são definidas com valores iguais para cada cor, assim CMYK 10,10,10,10 seria um tom cinza com 10% de preto, e 50,50,50,50 seria um cinza intermediário com 50% de preto.
