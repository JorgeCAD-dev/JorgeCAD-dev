Tratamento e Manipulação de Imagens

Para tratar, editar e manipular uma imagem utilizamos máscaras de seleção, que são modos de seleção onde parte dos pixels de uma imagem é selecionada, sendo então possível aplicar ajustes apenas sobre eles. Os softwares de edição de imagem contam com ferramentas criadas especialmente para o trabalho sobre pixels, alguns exemplos são:

Retoque

Ferramentas de retoque são ferramentas criadas para reconhecer um ponto de cor diferente e colori-lo com a cor dos pixels ao redor, geralmente utilizadas para remover imperfeições de pele.

Clone

Possui função semelhante, no entanto é possível escolher o local de onde será colhida a cor, e o local onde ela será aplicada, geralmente essas ferramentas são utilizadas para retoque e também para remover imperfeições, marcas dágua, textos e outros elementos indesejáveis de uma imagem.

Realce

Ferramentas de realce são utilizadas para aumentar o contraste entre as cores de um conjunto de pixels de uma parte da imagem onde há desfoque excessivo.

Desfoque

Ferramentas de desfoque são geralmente utilizadas para diminuir o contraste entre as cores de um conjunto de pixels.

Vale ressaltar que a depender do software utilizado as ferramentas utilizadas para cada uma dessas funções podem ter nomes diferentes.
Vale também ressaltar que cada software pode implementar mais ferramentas para diferentes funções.

Montagem

O processo de montagem consiste em recortar uma área selecionada de uma imagem e posteriormente reposicionar em outra imagem. É um processo extremamente útil quando possuímos uma foto de um objeto em fundo branco, preto, ou num croma-key e desejamos posicioná-lo em um fundo simulado específico, como, por exemplo, uma vitrine.

Filtros

Filtros são ajuste pré-configurados de cor, saturação, brilho e mesclagem utilizados para dar sensações especiais a uma determinada imagem, por exemplo um filtro vermelho aplicado a um pôr do sol pode aumentar a própria sensação transmitida pelo pôr do sol, ou um filtro azul numa imagem de geleiras ou de colinas nevadas pode aumentar a sensação de frio da paisagem.

Ajustes

Ajustes compõem parte fundamental da edição de imagens, a verdade é que muito raramente uma imagem é distribuída ou divulgada exatamente como ela foi tomada pelo fotógrafo ou renderizada pelo software. As imagens sempre passam por ajustes de brilho, contraste, matiz, saturação entre outros que visam corrigir valores ou acentuar certos aspectos.

Softwares de edição de Bitmaps

Existe uma grande quantidade de softwares de edição de imagens, como sempre, veremos as maiores alternativas comercial e open-source. Dentre as alternativas comerciais mais utilizadas atualmente com certeza o Adobe Photoshop ainda é o primeiro lugar da lista, por mais pesado que tenha ficado com tantos recursos que foram adicionados nos últimos anos, mesmo assim é o melhor software de edição e manipulação de imagens, sendo atualmente utilizado ainda para animação e pintura. Dentre as alternativas open-source, o primeiro lugar da lista é do GIMP (GNU Image Manipulation Program), um software completo para edição e manipulação de bitmaps com os recursos mais essenciais para qualquer designer gráfico ou fotógrafo.
