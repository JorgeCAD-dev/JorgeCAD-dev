O HTML

HTML é a abreviação de HiperText Markup Language que significa Linguagem de Marcação de Hipertexto, e como o próprio nome diz é uma linguagem de marcação assim como XML, utilizada para descrever páginas da web a serem exibidas em navegadores, e atualmente também para descrever interfaces utilizadas em aplicações móveis e desktop, com uso de “tempos de execução (runtime)” que permitem execução de páginas web fora do navegador.

Os arquivos HTML possuem a extensão .html e .htm e podem ser escritos em qualquer editor de texto, que serão interpretados pelo navegador. É uma linguagem que suporta:

  • Mídia: exibição de conteúdos de áudio, video, gráficos 2D e 3D.
  • Eventos: ações do usuário como clique, movimentação do mouse, uso de caneta e tela de toque.
  • Scripts e Estilos: Pode receber código em CSS para definição de estilo e Javascript para manipular lógica e alterar a estrutura do documento dinamicamente.
