Atributos Globais

Os atributos servem para definir propriedades e comportamentos de elementos HTML, entre eles está o atributo style que já utilizamos, mas existem vários outros, alguns atributos são específicos de um elemento HTML, outros são atributos globais que podem ser aplicados a uma grande variedade de elementos, a partir de agora veremos os principais.

title

O atributo title, muito utilizado, define o título que é exibido num tooltip próximo ao ponteiro do mouse: 

    
Um texto qualquer...

hidden

O atributo hidden é utilizado para ocultar o elemento HTML quando este não deve mais ser exibido na página, ele não precisa receber um valor e basta ser especificado: 

    
Um texto qualquer...

    
Um texto qualquer...

draggable

O atributo draggable é utilizado para transformar um elemento em “arrastável” definindo-o como true, sendo false o valor padrão: 

    
Isto é arrastável...

    
...isto não é!

lang

O atributo lang define o idioma de parte do texto de forma que o browser possa reconhecer em que linguagem está aquele texto fornecendo ferramentas de tradução: 

    
This is in english...

    
...En español...

    
...Em português do Brasil!

O idioma deve ser definido com um código ISO de idioma válido, se tiver dúvida confira em:

https://en.wikipedia.org/wiki/List_of_ISO_639-1_codes

style

O atributo style é utilizado para definir estilos CSS de elementos: 

    
Este texto é verde
