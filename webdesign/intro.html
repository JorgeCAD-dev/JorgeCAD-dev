Trilha de Aprendizado

O desenvolvimento para Web é um caminho longo e possui vários estágios, alguns obrigatórios, alguns opcionais de acordo com sua escolha profissional. Para facilitar as coisas para quem está entrando neste mundo, criamos uma trilha de aprendizado pronta. Para que você não fique perdido sobre o que deve aprender recomendamos que siga os cursos abaixo.

Lembrando que durante cada curso, várias explicações serão dadas para que você saiba perfeitamente a finalidade do que está aprendendo e para onde deve seguir depois. Este é o primeiro curso da Trilha de Desenvolvimento Web composta pelos cursos:

1. Web Design - HTML, CSS e Web Design Responsivo:

Crie sites estáticos profissionais e entre com o pé direito no mundo do Desenvolvimento Web.

2. Javascript:

Adicione lógica aos seus sites manipulando dados e ainda domine a linguagem de programação mais popular, capaz de praticamente tudo, de servidores à Inteligência Artificial.

3. Aplicações Web:

Aplicações Web com servidores para armazenar e tratar dados recebidos pelo site como cadastros e bases de dados. Aprenda ainda as tecnologias mais modernas da Web e torne suas aplicações disponíveis em lojas de aplicativos móveis e desktop.
