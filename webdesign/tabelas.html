Tabelas

Tabelas são componentes HTML com quatro níveis, a tabela em si, o corpo e o cabeçalho, as linhas e as células que juntas formam as colunas, uma estrutura de tabela é semelhante a esta: 

    

        
        
            

                Título 1
                Título 2
                Título 3
            

        
        
        
            

                Célula 1
                Célula 2
                Célula 3

A tag <thead> define o cabeçalho, onde estarão os títulos de cada coluna, estes títulos são defiinidos nas tags <th>. A tag <tbody> define o corpo da tabela, dentro do corpo inserimos as linhas <tr>, dentro de cada linha podemos inserir uma quantidade de células <td>, geralmente esta quantidade coincide com a quantidade de títulos inseridos no cabeçalho. Quando a tabela está montada, cada célula ficará abaixo de um título correspondente, formando assim as colunas.

Span

O span é utilizado para fazer com que uma célula ocupe duas ou mais linhas na vertical, ou duas ou mais colunas na horizontal, temos dois atributos de span que podemos aplicar a uma célula, colspan, o span de coluna e rowspan, o span de linha. Vejamos como funciona: 


    
        
Título 1
        Título 2
    

    
        

            Célula

Neste exemplo, a célula tem um colspan definido como 2, o que significa que ela ocupa sua própria coluna e a próxima coluna. Para que não haja um estouro da tabela é necessário que a próxima coluna não contenha uma célula, caso contrário o HTML criará mais uma coluna na tabela para acomodar todos os elementos. O mesmo é válido para o span de linha, vejamos um exemplo de tabela onde usamos todos os dois atributos:

            Título 3
