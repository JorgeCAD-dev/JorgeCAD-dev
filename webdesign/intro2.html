Introdução

O Web Design é o setor voltado para a criação de páginas Web Estáticas destinadas a exibir conteúdo, como sites, blogs entre outras, é um amplo setor e muito requisitado no mercado. No entanto, em módulos avançados aprenderemos também o desenvolvimento de aplicações web que gerenciam lógica e bancos de dados além de APIs da web.

Desenvolver uma apostila não é uma tarefa fácil, requer bastante conhecimento sobre o assunto e algum tempo de prática. Com a experiência isso se torna mais fácil. Mas desenvolver uma metodologia de ensino é bem diferente, a gente só aprende isso com experiência em sala de aula. Isso você não encontra no Youtube. Esse é o diferencial dos cursos disponibilizados por mim, uma metodologia de ensino que mescla experiência de sala de aula, experiência em escrita de livros (pois o autor é também escritor), e experiência no cuidado com os alunos. Tudo para garantir que ao término do curso não tenha apenas ocorrido um processo mecânico, mas para que o aluno possa começar do zero e sair dominando o assunto para se inserir no mercado de trabalho.

Estou constantemente desenvolvendo novos materiais e novos cursos, além de atualizar os materiais já produzidos, para acompanhar as alterações constantes nos produtos digitais. Portanto sugiro que esteja sempre atento em meu site para verificar se há algo novo que você goste, saiba mais.
