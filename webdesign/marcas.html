Marcas

O código HTML é definido com marcas, ou tags, que possuem abertura e fechamento:

<tag1>Conteúdo 1</tag1> <tag2>Conteúdo 2</tag2>

A tag define o tipo de conteúdo, uma lista completa com as tags HTML disponíveis pode ser encontrada em:

https://developer.mozilla.org/pt-BR/docs/Web/HTML/Element

Por exemplo, um título é definido com a tag h1:

<h1>Meu título</h1>

Um subtítulo:

<h2>Meu subtítulo</h2>

Um parágrafo:

<p>Algum texto...</p>

Um botão:

<button>Clique-me!</button>

Algumas tags não possuem fechamento, por exemplo, a tag <img> que insere imagem, e a tag <br> que insere uma quebra de linha num parágrafo, tags sem fechamento podem ser escritas assim:

<tag>

ou assim:

<tag/>

A barra utilizada no final da tag é opcional mas é utilizada como referência para sabermos onde está o final da tag, geralmente quando a linha se torna muito extensa.
